# musikoekosysteme-wien
# Musikalische Ökosysteme in Wien

Explorative Vorstudie über die räumliche Verteilung von Musikveranstaltungen 
und öffentliche Förderstrukturen im unabhängigen Musikfeld Wiens.

Gefördert von MA 7 – Kultur, Stadt Wien (2026).

## Status
In Bearbeitung / Work in progress

## Inhalt
- `dataset.csv` — Hauptdatensatz (Veranstaltungsorte, Bezirke, Förderverweise)

## Methodik
Datenerhebung aus öffentlichen Quellen. Analyse mit OpenRefine, Python und QGIS.
